## Parte 1: Investigación del Trabajo Soñado

### Perfil profesional elegido: Full Stack Developer

Un Full Stack Developer es un ingeniero capaz de trabajar tanto en el frontend como en el backend de una aplicación web, dominando toda la cadena de desarrollo desde la base de datos hasta la interfaz de usuario. Este perfil fue elegido por su alta demanda en el mercado laboral, versatilidad técnica y la posibilidad de construir productos completos de manera independiente.

## Perfil Profesional Objetivo

Con base en el análisis de las 5 ofertas laborales, el perfil profesional objetivo es:

> Full Stack Developer especializado en React + Node.js, con dominio de TypeScript, bases de datos relacionales (PostgreSQL), containerización con Docker y despliegue en la nube (AWS). Capaz de construir aplicaciones web completas, escalables y bien documentadas, con enfoque en buenas prácticas de desarrollo, testing y colaboración en equipos ágiles.

## Justificación del Enfoque del Sitio Web

El sitio web desarrollado sigue un enfoque de portafolio profesional orientado a postulación laboral, diseñado para demostrar las habilidades técnicas y proyectos relevantes para el perfil de Full Stack Developer.

- Las 5 ofertas coinciden en que se valoran proyectos demostrables con React, Node.js y bases de datos reales.
- El portafolio presenta exactamente el stack técnico más demandado: React, TypeScript, Node.js, PostgreSQL, Docker y AWS.
- La sección de proyectos muestra aplicaciones completas con frontend y backend, diferenciador principal del perfil Full Stack.
- El diseño es responsivo y moderno, reflejando dominio de CSS avanzado y buenas prácticas UI/UX.

## Diferenciadores Clave

1. **Dominio completo del stack moderno:** React + TypeScript + Node.js + PostgreSQL + Docker + AWS, que es exactamente lo que la mayoría de empresas demandan en 2025.

2. **Proyectos con impacto medible:** Los proyectos del portafolio muestran métricas de rendimiento y escala, conectando directamente con lo que los reclutadores valoran según el análisis de las 5 ofertas.
